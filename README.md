# CrcPik3caAllelicCounts

**CrcPik3caAllelicCounts** is an R package that provides allele-specific RNA and DNA read counts for **PIK3CA** mutations in colon cancer (CC) samples, derived from TCGA COAD data and curated for downstream mutant allelic expression imbalance (MutAEI) analyses.

## Contents

- `CRC_PIK3CA_AllelicCalls`: dataset containing allele-specific RNA/DNA counts per sample and mutation

- Documentation describing each variable and the data source

## Installation

### From GitHub (recommended)

```r
# install.packages("remotes")
remotes::install_github("aarbduarte/CrcPik3caAllelicCounts")
```
